Start using

Please register your account before using this system

Register account

  • IE9 or above is recommended to use or chrome browser to access mgedata
  • Register with your school email: Enter your username, password, confirmation password,email address, name (real name recommended), institution (school or company), all must be filled in and checked "I have read and agree to the Terms of Use and Terms and Conditions "Can be submitted
  • After registration,you will receive a notification in the message, please go to the mailbox to verify
  • If you have an account please log in directly to start using the system

Login account

If you already have an account, please enter your user name, password, and verification code. If you do not have an account, click "No account, register now"

  • If you do not fill in "username/email", the system will prompt "must fill in username"
  • If you do not fill in the "password", the system will be prompted "must fill in the password"
  • If you do not fill in the "verification code", a dialog box will pop up prompt "An error occurred, please fill in the verification code"
  • If you fill in the "username" and "password" are inconsistent, a dialog box will pop-up prompt " error, wrong password"
  • You can check to keep the login status, after opening this site you will be automatically logged in
  • If you forget your password, you can click "Reset Password" to set the password again

Personal settings

Settings are divided into personal information settings and avatar settings, click on the "personal information settings" to improve personal information and go to the mailbox click on the certification link to become certified users

Click "avatar settings", the right frame can be avatar adjustment, click to set the avatar, pop-up success is set to success

Templates

Templates introduction

Into the home page of "My Data", click on the template, select Add new entry, enter the template edit page

Or click the plus sign on the user's personal interface to enter the template creation interface directly

The template can only be created by the advanced user (database administrator and above), enter into the template creation page as follows

  • The submission template must be Authenticated and have the Admin and Above permissions or the dialog box will pop up "There was a problem with the template, non-accredited users could not create templates, please go to E-mail for certification "
  • You can choose to automatically save the template
  • After editing the template, you can choose to save and continue editing, or you can select "Create Template", the template is released, you can submit the template directly, and use the template to submit data, but if you do not click "Publish the template." The template will be Unpublished state, can not be found in the data submitted page just edit the template
  • Fill in the template title, the title can not be repeated, if repeated, the dialog box will pop up when submitted "Error, the template title can not be repeated, please modify and submit"
  • Select material classification, otherwise it will not submit and prompt "Please select material classification"
  • When you fill in the field name, do not repeat the name, if repeated, when submitted to the system will be marked red "field name can not be repeated"
  • The left side of the page is a template field type, when the mouse cursor touches the control, the system will give a message

Modify the templates

Into my data, click on the template, find the template you want to modify, click the modify button below, you can enter the modify interface

If the template has been submitted data, there will be a prompt "already have data using your template, modify the operation will create a new template, the original template will not be deleted, sure you want to modify it?", Select "OK" Enter the modification mode, then the name of the template will become "XXX-copy", the original template will not be deleted, still can be used if the template has not crossed the data, there will be no prompt, directly into the edit mode.

Upload data

Uploading data can be submitted in the following four ways.

  • Into the personal interface "My Data", select "Add New Item" to enter the upload data page
  • Click "View" in the home page and select "Upload" in the view page to enter the upload data page
  • Under the home page select "Upload" to enter the upload data page
  • Into the personal interface "My Data", click the upload arrow to enter the upload data page

After entering the submit page, select the material classification and the corresponding template, you can see there are 2 ways to submit: 1. Submit by form 2. Submit by file

Table submission

Click on the generated data form can generate the form to be filled in, fill in the content you want to and then it can be submitted, for example, as shown below

File submission

In the uploading interface, click "submit by file", you can choose to upload through Json, XML, Excel and other formats, according to the selected file, upload the corresponding type of file. In order to ensure that the template and upload the same file, please Advance to the template download page, according to the template prompts to fill in the data, and upload the final file.>

Excel submission

When uploading data via Excel, you need to have the following precautions:

  • For the data ID in the Excel table, the data ID of the same data needs to be the same, the value of the specific ID is optional, as long as the data ID is the same
  • If some of the data do not need to fill in, you need to empty the original prompt content in the template file to ensure that only the metadata remains in the table
  • Use Commas as a separator when filling in multiple files in a table , or range data
  • For the generator type, select which type to generate, the corresponding column to fill things , all other columns empty
  • Candidate data need to select the data in the template through the drop-down box in the Excel table
  • When you upload an attachment via Excel, the name of the attachment is best used in English, otherwise errors may occur, separated by commas when the same data contains more than one attachment
  • Fill in the data,and then you can put the picture and Excel in the same folder, compressed upload

Json submission

After entering the "Data Submission" page, select "File Submission". After generating the Json template, simply open the corresponding Json file, fill in the corresponding data to complete the generated Json file, upload the file, after the analysis is successful, you can view data

Data service

When the user first uses the system, a questionnaire will be conducted to know the user's interest and direction in material research. The user may also subscribe and the system will regularly recommend relevant data to the user.

Subscribe

  • At the end of registration, the relevant categories will be displayed for the subscribers.
  • In my subscription which you can choose to subscribe/unsubscribe categories
  • The news page displays the hot data based on the categories subscribed by the users.

Suggestion

  • Recommended content from the independent server analysis and access data derived and stored in the database
  • When the user request the recommended content, the system will read the database and get the recommended list

Feedback

Enter the home page, click on the "Aircraft" logo at the bottom right of the page, click will pop up a quick feedback box, there is a link next to the send button to enter the complete feedback page, with feedback pictures and other information can be completed through the feedback page

Field types

String type

A long list of texts, numbers, both Chinese and English can be entered

Numerical type

Arbitrary values, including integer, floating-point numbers, all of them can take units

Range type

A numerical interval, such as (a,b), or an error a ± b, a, b is numeric data

Candidate type

A series of candidates, the options can be grouped, temporarily allowed only one level, click Add Options, add one directly, click Add Group, the equivalent of adding a first level title, the group can continue to add a number of options.

Picture type

Common picture types, such as jpg, png and other formats, you can check the option "Allow multiple pictures" to allow multiple pictures to upload

File type

Common file types, such as pdf, word, excel, json.xml, etc. format, you can check the option "Allow multiple files" to allow multiple files to be uploaded

Array type

One-dimensional array, by selecting the array type, fill in the field name, click Set to this type to array name: [1,2,3,4,5,6,7,8]

Table type

Table, you can add string, numeric, range, picture, file type, candidate type of these six types of data, add a column, first select the column type, click the button "Add Columns" to add success

Container type

This type can contain all the types, can be any nesting, any type of field into the container-type box, you can nest successfully

Generator type

Name and type of the corresponding key-value combination, after the combination, by selecting a class to generate the corresponding form
